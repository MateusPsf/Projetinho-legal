# Calculadora Hematimétrica
.DS_Store
Ferramenta online para calcular índices hematimétricos baseado no hematócrito.

## Características.env
- Cálculo automático de índices hematimétricos
- Interface moderna e responsiva
- Fórmulas científicas aplicadas

## Como usar
1. Digite o valor do Hematócrito (Ht)
2. Veja os resultados em tempo real

## Fórmulas
- H = (Ht + 4) / 10
- Hb = Ht / 3
- VCM = (Ht × 10) / H
- HCM = (Hb × 10) / H
- CHCM = (Hb × 100) / Ht
